# 德州扑克模拟练习器

一个面向新手的德州扑克模拟练习器 MVP。目标是让初学者先建立三个核心直觉：牌力、底池赔率、位置，再逐步接入更专业的 GTO 复盘能力。

当前版本是纯静态前端，无需构建工具，可以直接打开 `index.html`，也可以用本地静态服务访问。

## 功能

- 6 人桌 No-Limit Hold'em 练习流程
- 多种简单 AI 对手风格：稳健、松凶、跟注、娱乐型
- 新手教练提示：当前牌力、底池赔率、位置意识
- 行动按钮：弃牌、看牌 / 跟注、加注
- 手牌记录和 JSON 导出雏形
- 后续接入 TexasSolver / postflop solver 的复盘接口预留

## 本地运行

直接双击 `index.html` 即可使用。

也可以启动本地静态服务：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 项目定位

这个项目目前不是完整 GTO 求解器。内置提示只用于帮助新手理解“为什么这个行动看起来更合理”，不是高精度策略表。

后续计划把练习器拆成三层：

1. 牌局引擎：使用 PokerKit 或后端服务替换前端内置牌局逻辑。
2. AI 对手：逐步加入基于规则、权益估算、强化学习或预训练策略的不同对手。
3. GTO 复盘：将牌局节点转换为 TexasSolver / postflop solver 输入，展示策略频率、EV 差距和更自然的新手解释。
